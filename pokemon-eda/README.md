# Pokemon EDA Project

Bu projede Pokemon veri seti üzerinde keşifsel veri analizi (EDA) yapılmıştır.

## Amaç
- Pokemon istatistiklerini incelemek
- Türlere göre güç dağılımını analiz etmek
- Veri görselleştirme pratiği yapmak

## Kullanılan Araçlar
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dosyalar
- data-science-pokemon.ipynb : Analiz notebook’u
